jsFiddle: an online playground

Code and render HTML, scripts and css within browser.

It gives you a quick way to test and tweak code without having to switch back and forth between editor and browser, or upload code to a server to run.

http://jsfiddle.net

Stunning examples

JsFiddle is a super slick website that let you edit and render HTML, scripts and css online. It gives you four panes. One for JavaScript, one for HTML, one for CSS and one to preview your code as it would run in a browser. [Example: jQuery Delay, Run.] [Stunning examples -> Run/Edit] Clicking "Edit" will load the scripts html and css in jsFiddle, so everything is there for you to tweak and make it your own. The beauty of this is quick prototyping: You rip off an examples page, an instant base for trying out your own ideas.
