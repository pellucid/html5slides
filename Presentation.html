jsFiddle: a online playground

http://jsfiddle.net/r7MgY/

http://andrewwooldridge.com/blog/2011/03/16/stunning-examples-of-using-jsfiddle/

Renders scripts, HTML and css on the fly.
JsFiddle is an amazing tool that is accessible anywhere via a browser. It gives you four panes. One for JavaScript, one for HTML, one for CSS and one to preview your code as it would run in a browser. This gives us a great way to test out and tweak code on the fly without having to switch back and forth between editor and browser, or upload code to a server to run. The beauty of JSFiddle is quick prototyping: You rip off some script, html and css from an examples page, and tweak it and make it our own.
